Ko'proq

Sikl qanday ishlaydi

JavaScriptda sikl (yoki loop) – bir amalni yoki kod blokini qayta-qayta bajarish imkonini beruvchi konstruksiya. JavaScriptda eng ko‘p qo‘llaniladigan sikl turlari: for, while, va do...while.

Sikl yordamida massiv elementlarini o‘qish, hisoblash amallarini takrorlash, yoki ma'lum shart bajarilgunga qadar jarayonni davom ettirish mumkin.

Ko'proq

For


// for (let i = 1; i <= 5; i++) {
// console.log(i);
// }

Tushuntirish:
1. let i = 1; - Bu boshlang'ich qiymat qadamidir. Sikl boshida i ni 1 ga tenglaydi.
2. i <= 5; - Bu shart qismidir. Sikl i 5 dan kichik yoki teng bo'lganda davom etadi.
3. i++ - Bu yangilash qadamidir. Har bir aylanishdan keyin i qiymati 1 ga oshadi.

Natija: 1 2 3 4 5

While


// let i = 1;
// while (i <= 5) {
// console.log(i);
// i++;
// }

Tushuntirish:
1. let i = 1; - Bu boshlang'ich qiymat. i ga 1 qiymati beriladi.
2. while (i <= 5) - Bu shart qismi. Sikl i 5 dan kichik yoki teng bo'lganda davom etadi.
3. i++ - Bu yangilash qismi. Har bir aylanishdan keyin i qiymati 1 ga oshadi.

Natija: 1 2 3 4 5

Do while


// let i = 1;
// do {
// console.log(i);
// i++;
// } while (i <= 5);

Tushuntirish:
1. let i = 1; - Bu boshlang'ich qiymat qadamidir. Sikl boshida i ni 1 ga tenglaydi.
2. do { ... } - Bu sikl tanasi. Kod blok ichida joylashgan kod kamida bir marta bajariladi.
3. while (i <= 5); - Bu shart qismi. Sikl i 5 dan kichik yoki teng bo'lganda davom etadi.

Natija: 1 2 3 4 5
